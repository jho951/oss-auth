## 읽는 순서

- 공개 설정 계약은 `oss-contract` 저장소를 봅니다.
- 처음 사용하는 사람은 `아키텍처`, `모듈 가이드`, `SPI/extension 가이드` 순서로 보면 됩니다.
- 테스트를 돌리거나 publish 흐름을 확인할 때는 `테스트/CI 가이드`를 봅니다.
- SPI를 직접 구현하는 경우 `SPI/extension 가이드`를 먼저 보세요.
